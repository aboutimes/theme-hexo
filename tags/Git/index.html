Git 笔记

2017-08-20 | by COUH


　　Git 是一个开源的分布式版本控制系统，用以有效、高速的处理从很小到非常大的项目版本管理。说白了就是一个版本控制工具，帮助大家管理自己的代码。Git 最初是 Linus 为了帮助管理 Linux 内核开发而编写，取得良好效果，之后向社区开源，于是就有免费而超级好用的 Git 。
