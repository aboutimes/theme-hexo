我曾用过的那些手机

2020-04-12 | by COUH


　　前段时间老爸突然来办公室，也没提前招呼，挺吃惊，还以为是来办事的群众，一说才知道是手机坏了，刚好有一部扶贫用的安卓机，顺手就给他了。之后这段时间经常下乡扶贫，没有安卓手机确实不方便，所以就想着重新添置一部，咸鱼逛了圈，发现了 Pixel 3 ，挺对味，刚好又想体验一下传说中谷歌相机，所以就有了第一部大屏手机。

1、Pixel 3


　　手机很快到手，手机很薄，手感不错，但是确实大，用惯了 4 英寸屏，一下子切换到 5.5 英寸，感觉太大了，单手操作不方便，而且原版居然不能使用电信 4G ，好在可以破解，一番折腾解锁，总算可以正常使用。虽然屏幕是 2160×1080 的 OLED 屏，但是颗粒感挺严重，没有手上 iPhoneSE 细腻。苹果系统用久了，突然切换到安卓系统确实有些不习惯，安卓联动功能没有苹果好用，每次想共享图片到电脑都很麻烦，不像苹果系统有开箱即用的隔空投送。另外是很多优质的应用没有安卓版，或者是安卓版体验下降，最明显的区别就是应用风格与系统割裂。安卓优点是更开放，软件安装更自由，不用担心安装了新版本体验达下降却无法回退，其次是文件管理更自由，方便分类管理。另外，谷歌相机拍照是真的好，特别是高光比场景的宽容度真是让人吃惊，甚至与手里 APSC 画幅的 GR2 相机持平，但是也有一些小缺点，即使是光线好的情况下照片也有些明显噪点。

2、iPhone


　　初次接触苹果手机是在 2014 年初，那会儿还没开始记笔记，所现在也记不起当初为什么会买苹果的手机了，也许是图个新鲜，在 MACX 论坛上买了台二手 iPhone 5，也不知为何，自此之后买手机就都是买二手，也是视觉的全新溢价太高，还有就是喜欢折腾的人对自己技术的信任吧。然后就接触了 Apple ID ，iOS，iCloud，自此爱上苹果手机系统，简洁实用，不折腾。后来折腾了黑苹果，感觉像打开了新大门，集苹果手机系统的简洁和安卓系统的开放性于一身，体验到苹果手机和苹果电脑的联动性，便对苹果生态更加迷恋和依赖，想着下一台笔记本一定是 Macbook Pro 。不过也许是苹果电脑系统比较耐用，直至升级了 iPhone SE ，再到 Pixel 3，当年的那台笔记本依旧服役，其中一个原因是当年换的华硕 UX301 比较贵，舍不得淘汰这么快，而且很难再找到这么符合心意的笔记本了，另外就是没有很大的更换动力，这几年接口改动过于激进，键盘又是问题不断，一直在等下一代。

3、st8i

3.3 英寸 480×854像素(FWVGA)多点触控屏
索尼 Mobile BRAVIA Engine(手机图像处理引擎)
No air-gap 技术（无缝隙屏显科技）
810 万摄像头背照式 CMOS
高通 MSM8255 1GHz 处理器
512MB RAM + 1GB ROM 内存组合
Adreno 205 GPU

　　索尼爱立信 st18i 是使用的第一部智能手机，2012 年工作之后第二个月入手，现在都佩服自己当时能选到这么经典的手机，造型小巧优美，配置强劲，索爱小绿标。由功能机切换到智能机，真的是质的飞跃，像打开了新世界。也是这部手机开启了我的折腾之旅，root 之后大部分系统应用都进行过修改。最喜欢的是索爱独占应用 TrackID 可以识别歌曲，类似于微信摇一摇搜歌，不过识别率好太多，这在当时简直是神器，不过现在这个应用停止服务了，非常可惜。信息小组件可视化效果很棒，类似 win7 的3D窗口切换界面。比较遗憾的是某次手抖把硬盘格式化，导致这部手机的资料都丢失了，好多珍贵的照片也就此丢失。如果这手机硬件能升级到主流配置，充电口再改到底部，就是理想的完美手机。

4、Nokia X2

2.2英寸 240×320(QVGA)像素 TFT
S40 系统
48MB 内存
500 像素 CMOS

　　Nokia X2 是第一部自购的手机，购买主要原因是大三去洛阳实习时手机丢了，那是一台更古老的诺基亚，虽然如此，当时丢了还是挺伤心的，回来之后就买了诺基亚 X2。Nokia X2 搭载 S40 系统，破解之后可以后台挂 QQ ，在当时手机主要是通讯工具，这已经够用。另一个喜欢的功能是不用插耳机也能听广播，很实用，现在的手机基本已经去除了收音功能。自然也少不了经典的贪吃蛇，当时热门手游之一，此外还可以安装推箱子、实况足球、赛车等游戏。十年时间，变化真大，从功能机到智能机，再到智能系统之争，再到大屏革命，手机的更换历史伴随着手机技术革命史，不变的是一直钟爱小屏手机。
